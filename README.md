kindness
========

Some cool stuff for awesome wm.

## How to install
You can download archive and extract to awesome lib derictory (```/usr/share/awesome/lib```), lua lib derictory (```/usr/share/lua/5.2```) or your config derictory.

## What's in the box?!
* Slider widget
* Scroll layout
* Relative layout
* Animation (experimental)
